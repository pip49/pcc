PC정비사 2급 필기 기출문제 요약

2024년 08월 25일 시행 | 오개념 방지 교정 버전
1과목: PC유지보수
1. 합법적으로 소유하고 있던 사용자의 도메인을 탈취하거나 도메인 네임 시스템 (DNS)의 주소를 변조함으로써 진짜 사이트로 오인해 접속하도록 유도한 뒤 개인 정보를 훔치는 컴퓨터 범죄 수법은?
파밍 (Pharming)
2. 단편화 현상 중, 프로세스들의 가변 크기 파티션 할당 및 해제 주기가 반복되면서 메모리 사이사이에 생겨나는 매우 작은 조각 공간들로 인해 발생하는 단편화의 종류는?
외부 단편화
3. Windows 10 Pro 환경에서 가상 메모리를 수동 설정 및 제어할 때 운영체제 UI단에서 기본적으로 명확히 제공해 주는 정보 항목 3가지는?
드라이브[볼륨 레이블], 모든 드라이브의 총 페이징 파일 크기, 선택된 드라이브의 페이징 파일 크기 (※ 세그먼트의 크기는 가상 메모리 설정창에서 제공되지 않음)
4. 롬(ROM) 반도체 미디어 종류 중 하나로, 내부 칩 표면에 쿼츠 창이 달려있어 입력된 내용을 특수 자외선(UV) 레이저로 깨끗이 지우고 새로운 바이너리 코드를 다시 입력할 수 있는 롬 명칭은?
EPROM (Erasable Programmable ROM)
5. Windows 10 Pro 시스템 구성 제어 도구(msconfig.exe) 내부의 [도구] 탭 메뉴를 통해 직접 논스톱 실행할 수 있는 핵심 하위 콘솔 기능들은 무엇인가?
이벤트 뷰어, 성능 모니터, 시스템 정보 등 (※ 네트워크 파일 보관을 다루는 '동기화 센터'는 하위 메뉴에 없음)
6. Windows 10 환경에서 네트워크 어댑터 및 원격 통신 진단을 수행하는 주요 네트워크 명령어들의 올바른 공학적 사실 정의는?
'netstat -an'(내 PC에 연결된 상대방 IP 확인), 'ping 내 컴퓨터 IP'(네트워크 카드 하드웨어 정상 여부 확인), 'tracert'(상대방 컴퓨터까지 통신 패킷 경로 추적) / ※ nslookup은 도메인명과 IP 주소 간 상호 맵핑 정보를 DNS에 질의하는 명령어임
7. 컴퓨터 아키텍처 연산 처리 동작 중 예외 및 가로채기를 유발하는 인터럽트(Interrupt)에 관한 실질적 기술 진실은?
정전 발생, 입출력 종료 및 오류 발생, 허용되지 않은 기억 공간 무단 접근 시 발생하며, 사용자가 시스템 콜을 유발하는 소프트웨어 인터럽트를 의도적으로 직접 발생시키는 것도 구조적으로 가능하다
8. Windows 운영체제 환경에서 시스템 레지스트리 편집기(regedit)를 조작하여 설정을 편집할 때 유의해야 할 필수 엔지니어링 지식은 무엇인가?
편집 직전 무조건 백업을 수행하고, 키 삭제 전 미리 내보내기 저장을 해야 하며, 수정 즉시 값이 시스템에 반영됨을 인지해야 한다. (※ Windows 레지스트리 키 및 값 이름은 시스템상에서 '대소문자를 구분하지 않음')
9. 가상 메모리(Virtual Memory) 관리 아키텍처 환경을 운용할 때 내부 커널 및 가상 주소 변환 과정에서 유기적으로 사용되는 핵심 핵심 개념 및 용어 3가지는?
페이지 폴트(Page Fault), 디맨드 페이징(Demand Paging), 스래싱(Thrashing) (※ DMA는 CPU 개입 없이 장치와 RAM이 다이렉트로 데이터를 전송하는 하드웨어 제어 기술임)
10. 데이터 전송 도중 노이즈 등으로 인해 에러가 발생했을 때, 수신측 시스템이 단순히 오류를 검출하는 데 그치지 않고 코드 규칙을 이용하여 스스로 오류 비트 위치를 찾아내어 직접 정정(수정)까지 완벽하게 수행할 수 있는 특수 코드는?
해밍 코드 (Hamming Code)
11. Apple 사가 매킨토시 하드웨어 전용으로 탑재하여 배포 및 발전시켜 온 전용 독점 운영체제인 macOS의 정밀한 아키텍처적 근본 계보는?
Unix(유닉스) 커널 아키텍처를 기반으로 개발 및 설계되었다
12. Windows 10 환경에서 로그인 암호, PIN 번호, 생체 인식(Windows Hello) 등 사용자 계정의 암호 설정을 직접 변경하거나 추가 제어하고자 할 때 선택해야 하는 세부 설정 카테고리 명칭은?
로그인 옵션
13. Windows 운영체제에서 텍스트나 파일을 복사(Ctrl+C)하거나 잘라내기(Ctrl+X)를 실행했을 때, 다른 곳에 붙여넣기(Ctrl+V) 하기 전까지 시스템 주기억장치 상에 임시로 보관 처리를 전담하는 특수 메모리 영역 명칭은?
클립보드 (Clipboard)
14. 로컬 PC에 유효하고 원활한 인터넷 통신망(TCP/IP 프로토콜) 환경 구성을 구현하기 위해 네트워크 관리자가 직접 주소 번지를 배정하거나 설정해야 하는 3대 필수 파라미터는?
IP 주소, 서브넷 마스크, 게이트웨이(Gateway) 주소 (※ 최대 소켓의 수는 운영체제 커널 내부 커넥션 자원 한계치일 뿐, TCP/IP IP 기본 구성 입력 항목이 아님)
15. 새 하드디스크나 SSD를 구매하여 포맷을 완료한 후, 사용자가 운영체제 상에서 인식하는 실제 가용 용량이 제조사 포장 패키지에 기재된 표기 용량보다 현저히 적게 나타나는 명확한 컴퓨터 공학적 이유는?
하드디스크 제조업체는 10진수 계산법을 써서 1KB를 1000Byte로 계산하여 생산하지만, 컴퓨터 운영체제(OS)는 2진수 구조를 쓰기 때문에 1KB를 1024Byte로 나누어 환산 인식하기 때문이다
2과목: PC운영체제 및 하드웨어
16. 메인보드에 결착되는 시스템 메모리(RAM)의 연산 제어 지표인 '메모리 타이밍(Memory Timing)'에 대한 정확한 하드웨어 이론 지식은 무엇인가?
메모리 내부에서 데이터를 얼마나 빨리 찾는지를 규정하는 제어로 숫자가 작을수록 속도가 빠르고 tRAS 등이 대표적이며, 오버클러킹 및 메모리 튜닝 성능 전반에 지대한 영향을 미친다
17. 롬(ROM) 반도체 칩이 메인보드 제어 버스 회로와 연동하여 동작할 때, 연산 상태 및 처리 방향을 제어하기 위해 인입 및 출력되는 주요 핵심 전기 제어 신호 종류는?
READ(읽기 신호), WRITE(쓰기 신호 - EEPROM/플래시 등), CHIP SELECT(칩 선택 활성화 신호) / ※ DATA BUS는 제어 신호가 아닌 실제 패킷 데이터가 지나다니는 물리적 통로 전선임
18. 자기 디스크 매체인 하드디스크 자성 표면에 디지털 0과 1 데이터를 물리적 자기 신호로 배열하여 안전하게 기록·보관하기 위해 사용하는 실질적 내부 자화 물리 저장 방식 종류 3가지는?
NRZ(Non Return to Zero), MFM(Modified Frequency Modulation), RLL(Run Length Limited) (※ IDE는 하드디스크를 메인보드에 결선하는 통신 인터페이스 버스 규격명임)
19. 과거 병렬 포트(LPT) 기반 인쇄 장치 연결 시 규정되어 사용되던 IEEE 1284 표준 프린터 데이터 고속 전송 모드 프로토콜 종류는?
SPP(표준 병렬), EPP(향상된 병렬), ECP(확장 기능 포트) (※ MPP는 프린터 병렬 규약에 존재하지 않음)
20. 컴퓨터 시스템 하드웨어 구조에서 보조기억장치(HDD, SSD 등)가 가지는 전형적인 장치 물리적 아키텍처 특징은 무엇인가?
중앙처리장치(CPU)와 내부 다이렉트 자원 교환이 불가능하여 주기억장치를 거쳐야 하므로 접근 시간(Access Time)이 상대적으로 크며, 데이터 전송 최적화를 위해 주로 DMA 방식을 사용한다
21. 동기식 고속 메모리 반도체 아키텍처인 DDR(Double Data Rate) RAM 기술이 기존 SDRAM 대비 데이터 대역폭을 2배로 획기적으로 끌어올릴 수 있었던 물리적 핵심 연산 메커니즘은 무엇인가?
시스템 클록 주파수가 진동할 때, 클록의 상승 에지(Rising Edge) 시점뿐만 아니라 하강 에지(Falling Edge) 시점에도 데이터를 정합성 있게 동시에 실어 나르는 전송 기술을 적용했기 때문이다
22. 액정 디스플레이(LCD) 모니터 패널 종류 중 하나로, 분자 배열 전환이 매우 빨라 잔상이 적고 응답 속도가 대단히 우수하며 제조 단가가 저렴하지만, 측면이나 위아래에서 볼 때 색이 왜곡되는 시야각 협소 단점이 있는 패널은?
TN (Twisted Nematic) 패널
23. 컴퓨터 본체 구동 시 프로세서의 연산 데이터를 실시간으로 보관 및 적재하는 주기억장치 계열 중, 전원이 차단되면 내부 데이터가 즉시 소멸하는 순수 물리 휘발성 램(RAM) 제품 소자 종류는?
DRAM, SDRAM, SRAM (※ 플래시 메모리는 전원이 꺼져도 데이터가 유지되는 대표적인 비휘발성 반도체임)
24. 유저가 화면 좌표를 제어하는 주변기기인 마우스(Mouse) 장치를 컴퓨터 본체 메인보드 칩셋 단에 결선하여 가동하기 위해 역사적 및 현재적으로 지원되어 온 통신 인터페이스 포트 종류 3가지는?
시리얼(Serial) 포트, PS/2 포트, USB 포트 (※ IDE는 구형 하드디스크 및 ODD 전용 병렬 데이터 케이블 규격임)
25. 기술자가 PC 본체 케이스 내부를 열어 대용량 스토리지 장치(하드디스크 및 SSD)를 메인보드 인입 회로망과 데이터 통신이 가능하도록 마운트할 때 사용하는 올바른 인터페이스 버스 규격 종류는?
SATA, E-IDE, SCSI 등 (※ DP는 모니터 화면에 고해상도 영상 및 음성을 송출하는 그래픽 디스플레이 포트 규격임)
26. 문서 및 사진을 디지털화하는 스캐너(Scanner) 기기 제원 및 성능 지표에 명시되는 '해상도(Resolution / DPI)'의 정확한 공학적 기술 의미는?
해당 스캐너 장치가 대상 종이 문서를 얼마나 미세하고 정밀한 격자 점으로 쪼개어 스캔할 수 있는지, 즉 '스캔할 수 있는 이미지의 품질(정밀도)'을 나타낸다
27. 스마트 키오스크나 터치 모니터 기판 표면에 탑재되어 유저의 신체 접촉 입력을 검출해 내는 터치스크린 장치 중, 실제 내장되는 '물리 센서 구동 기술 방식'에 따른 올바른 공학적 분류 3가지는?
초음파(SAW) 방식, 정전용량(Capacitive) 방식, 적외선(IR) 방식 (※ 단일터치는 센서 종류가 아니라 동시 인식 개수에 따른 소프트웨어적 기능 분류임)
28. 초고속 데이터 전송 및 고출력 전력 송전을 동시 허용하는 24핀 구조의 최신 표준 단자로, 커넥터 상하 단면 구조가 완벽한 대칭 형태로 설계되어 유저가 방향을 확인하지 않고 거꾸로 꽂아도 정합성 있게 연결되는 USB 포트 명칭은?
USB Type-C
29. 반도체 기반 스토리지 장치인 SSD(Solid-State Drive)가 기존 자기 플래터 방식 하드디스크(HDD) 대비 가지는 강력한 하드웨어적 구조적 장점 특성은?
부팅 및 로딩 속도가 획기적으로 빠르고, 물리적 헤드가 없으므로 진동과 충격에 강해 자료 보호에 유리하며, 고화질 영상 편집 속도를 단축시킨다. 또한 전력 소비량이 매우 낮아 노트북 배터리 유지 효율을 극대화한다.
30. 하드디스크 디바이스 내부 플래터 미디어가 물리적으로 저장할 수 있는 총 데이터 보관 용량(Total Capacity)을 산출해 내는 정밀 공학적 물리 계산 공식 구문은?
헤드 수 × 실린더 수 × 섹터 수 × 섹터당 바이트 수
3과목: PC주변기기 및 안전관리
31. PC 메인보드 상에 장착되어 본체 인입 전원이 완벽히 차단된 대기 상태에서도 컴퓨터 내부 물리 하드웨어의 로우레벨 설정 사항, 자원 구성값, 현재 시스템 시간 데이터를 지속해서 보관 및 유지하고 있는 장소는?
CMOS (Complementary Metal-Oxide Semiconductor) 램 영역
32. 조립 컴퓨터 빌드 시, 최고 전력 상태의 고부하 CPU 반도체 다이 표면에서 발생하는 극심한 열을 상변화 냉각액과 전용 펌프 모터를 이용해 고속 순환시켜 방열하는 '수냉식 쿨러 키트'를 구성하는 핵심 필수 파트 부품 종류는?
라디에이터, CPU 워터블럭, 써멀 컴파운드 (※ 거대한 알루미늄 방열판 패널과 냉각팬 위주로 짜인 'CPU 쿨러 타워형'은 전통적인 공랭식 쿨러의 핵심 부품임)
