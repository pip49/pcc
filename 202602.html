PC정비사 2급 필기 기출문제 (2026년 02월 01일 시행)

1과목 : PC 유지보수

1. Windows 11 환경에서 파일 내용, 수정 날짜, 파일 유형 등을 바탕으로 AI 기반 통합 검색을 수행하고자 할 때 가장 적합한 기본 기능은?
정답: 작업 표시줄의 '검색(Windows Search)' 기능
2. 노트북이나 PC 화면을 회의실 TV 등 '무선 디스플레이에 연결'하여 화면을 송출할 때 사용하는 Windows 단축키는?
정답: Windows 로고 키 + K
3. Windows 11 환경에서 스마트폰 연동 또는 외부 무선망을 이용하기 위해 셀룰러 데이터 연결을 설정하는 올바른 기본 설정 경로는?
정답: 설정 > 네트워크 및 인터넷 > 셀룰러
4. Windows 11 시스템의 가상 메모리(Paging File) 설정 창에서 기본적으로 확인 및 제공하는 정보들을 나열하시오. (부정형 변환)
  • 드라이브 [볼륨 레이블]
  • 모든 드라이브의 총 페이징 파일 크기 정보
  • 선택된 드라이브의 페이징 파일 크기 지정 정보
5. 사용자가 컴퓨터를 장시간 비우거나 절전을 도모할 때, 작업 중인 메모리 데이터를 스토리지의 특정 영역(hiberfil.sys)에 안전하게 대피 및 저장하고 시스템 주 전원을 차단하여 전력 소모를 완전히 억제하는 모드는?
정답: 최대 절전 모드 (Hibernation)
6. Linux 운영체제에서 특정 파일의 내부 텍스트 콘텐츠 데이터나 팩트 정보를 화면상에 다이렉트로 출력 및 확인하고자 할 때 사용하는 대표적인 명령어는?
정답: cat
7. Windows 11의 명령 프롬프트(CMD) 환경에서 지정한 하드디스크 볼륨 드라이브 내부의 단편화 상태를 종합적으로 분석하고 최적화하여 액세스 속도를 복원 및 향상시켜 주는 도구의 명령어는?
정답: defrag
8. Linux / Unix 커널 환경에서 현재 시스템 내부 백라운드 및 포어그라운드에서 동작 중인 모든 활성 프로세스들의 유기적인 상태와 트리 흐름 구조를 실시간 확인하는 표준 명령은?
정답: ps
9. 운영체제가 기억장치를 효율적으로 제어하기 위해 주기억장치를 고정 크기 블록으로 분할하여 매핑 및 처리하는 페이징(Paging) 기법의 핵심적이고 올바른 사실은 무엇인가?
정답: 외부 단편화는 완전히 방지되나, 마지막 블록 크기 미달로 인한 내부 단편화(Internal Fragmentation)가 미세하게 발생할 수 있다.
10. Windows 11에서 하드디스크의 논리적 논리 구획 단위인 볼륨의 오류 상태를 정밀 진단하고, 배드 섹터 체크 및 파일 시스템 논리 오류를 수정 및 치료해 주는 복구 도구 명령어는?
정답: chkdsk

2과목 : PC 운영체제 및 주변기기

11. 메인보드 하드웨어 슬롯에 결합하여 CPU 연산 처리를 보완하는 RAM의 한 종류로, 동기식 시스템 버스 구조와 결을 같이하여 매 클록 주기의 상승 에지와 하강 에지에서 모두 데이터를 동시 전송해 처리 효율을 극대화한 아키텍처는?
정답: DDR SDRAM (Double Data Rate SDRAM)
12. CPU의 물리적 핵심 명세 중 하나인 내부 캐시 메모리(L1, L2, L3)에 채택되어, 전원이 인가되는 동안 데이터가 리프레시 동작 없이도 영구 유지되고 연산 속도가 극도로 빠른 메모리 반도체 소자는?
정답: SRAM (Static RAM)
13. 모니터 인터페이스 중 디지털 영상 신호와 디지털 오디오 신호를 단 하나의 통합 케이블 라인에 완벽히 결합하여 무손실로 디스플레이에 전송해 주는 영상 출력 포트 표준 규격은?
정답: HDMI (High-Definition Multimedia Interface)
14. CPU의 구조 유형 중 명령어 세트를 단순화 및 최적화하여 연산 속도를 획기적으로 향상시켰으며, 모바일 AP 및 임베디드 저전력 고효율 칩셋의 기본 아키텍처가 되는 설계 방식은?
정답: RISC (Reduced Instruction Set Computer)
15. UTP 케이블 제작 시 전자기적 간섭(EMI)과 신호 겹침 현상인 누화(Crosstalk) 오류를 예방하기 위해 케이블 물리 구조 내부에서 채택하고 있는 핵심적인 특징은?
정답: 구리선 두 가닥을 서로 일정한 주기와 횟수로 배배 꼬아놓은 트위스티드 페어(Twisted Pair) 결선 구조
16. 비휘발성 플래시 메모리 소자를 활용하여 물리적 모터 구동 소음이 전혀 없고 고속의 데이터 읽기/쓰기가 가능해 현대 PC의 메인 스토리지 표준으로 정착한 보조기억장치는?
정답: SSD (Solid State Drive)
17. 구형 그래픽 카드와 모니터를 연결할 때 아날로그 비디오 신호만을 전송하는 포트 구조로, 총 15개의 핀 배열로 이루어진 레거시 영상 출력 포트 규격 명칭은 무엇인가?
정답: D-Sub (또는 VGA 포트)
18. 컴퓨터의 심장인 파워서플라이(Power Supply) 유닛이 공급하고 조율하는 직류(DC) 출력 전압 규격 종류들을 모두 나열하시오. (부정형 변환)
  • +3.3V (메인보드 로직 회로 및 칩셋 구동용)
  • +5V (스토리지 컨트롤러 및 USB 포트 공급용)
  • +12V (CPU 및 외장 그래픽카드 보조 전원 구동용)

19. 조립 컴퓨터 유지보수 시 하드웨어 슬롯 접촉 불량으로 인한 비프음 오류를 해결하기 위해 메인보드 램 슬롯이나 카드 접점 부위의 미세 이물질 및 산화막을 세척할 때 쓰는 유용한 화학 용품은?
정답: 접점부활제 (Contact Cleaner)

정답: 모니터 신호 케이블(HDMI/DP)의 결합 상태 및 본체 외장 그래픽카드(VGA)의 슬롯 안착 상태
