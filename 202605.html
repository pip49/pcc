PC정비사 2급 필기 기출문제 (2026년 05월 17일 시행)

1과목 : PC 유지보수

1. Windows 11의 핵심 시스템 파일이 아닌 대표적인 일반 응용 프로그램 실행 파일은 무엇인가? (부정형 변환)
정답: excel.exe (엑셀 프로그램은 운영체제 핵심 커널 및 서브시스템 파일이 아닌 오피스 응용 소프트웨어임)
2. Windows 11 환경에서 파일 탐색기가 '응답 없음' 상태로 멈췄을 때, 재부팅 없이 명령 프롬프트(CMD)를 통해 탐색기 프로세스만 강제 종료하고 다시 안전하게 재시작하는 명령 조합은?
정답: taskkill /f /im explorer.exe 실행 후 start explorer.exe 입력
3. Windows 11 작업 표시줄이나 시작 메뉴의 앱 아이콘을 마우스 우클릭했을 때, 최근에 열었던 파일 목록이나 자주 사용하는 항목 및 핵심 작업을 바로 실행할 수 있도록 지원하는 메뉴 기능의 명칭은?
정답: 점프 목록 (Jump List)
4. Windows 11 핵심 기본 시스템 프로세스(예: lsass.exe, smss.exe 등)의 주된 역할은 무엇인가?
정답: 로컬 보안 인증 하위 시스템(LSASS) 제어 및 세션 관리자(SMSS)를 통한 초기 부팅 환경과 사용자 세션 생성
5. Windows 11의 전원 설정 메뉴에서 지원하는 디스플레이 및 시스템 대기 모드로, 작업 상태를 RAM에 유지하되 전력 소모를 일시적으로 최소화하여 신속히 이전 상태로 복귀할 수 있게 돕는 모드는?
정답: 절전 모드 (Sleep Mode)
6. Linux / Unix 시스템 아키텍처 환경에서 특정 텍스트 파일의 전체 줄 수(Line), 단어 수(Word), 바이트/글자 수(Byte)를 정밀하게 계측하여 통계를 보여주는 표준 명령어는?
정답: wc (Word Count)
7. Windows 11에서 모니터의 화면 배율, 텍스트 크기 조절, 다중 디스플레이 구성 및 화면 해상도를 설정하기 위해 진입해야 하는 올바른 Windows 설정 제어 메뉴 항목은?
정답: 시스템 (설정 > 시스템 > 디스플레이)
8. Linux / Unix 운영체제에서 백그라운드나 포어그라운드로 동작하고 있는 특정 특정 프로세스를 식별자 고유 번호(PID)를 지정하여 강제로 즉각 종료하고자 할 때 사용하는 표준 명령은?
정답: kill
9. 운영체제 메모리 관리 전략 중, 프로세스가 필요로 하는 페이징 공간을 디스크 보조기억장치로부터 선제적으로 호출하고 실시간 적재하여 효율을 추구하는 반입(Fetch) 전략 기법은 무엇인가?
정답: 요구 반입 (Demand Fetch)
10. Windows 11 명령 프롬프트(CMD) 환경에서 윈도우 기본 시스템 파일의 손상 여부를 정밀 무결성 검사하고, 손상이나 유실이 발견될 경우 자동으로 안전하게 복구 및 치유해 주는 시스템 관리 명령어는?
정답: sfc /scannow

2과목 : PC 운영체제 및 주변기기

11. 메인보드 펌웨어 영역에 장착되어 PC 전원이 인가된 직후 CPU, RAM, 그래픽카드 등 주변 하드웨어 장치에 전기가 안정적으로 들어왔는지 스스로 정밀 진단 및 하드웨어 점검을 실행하는 프로세스는?
정답: POST (Power On Self Test)
12. PC 및 멀티미디어 환경에서 외부 가전 장치나 악기 및 오디오 믹서를 컴퓨터 사운드 장치와 연결하여 정밀 디지털 사운드 데이터 및 제어 신호를 상호 교환하기 위해 정의된 인터페이스 규격은?
정답: MIDI (Musical Instrument Digital Interface)
13. 모니터 디스플레이 인터페이스 중 순수 디지털 영상 및 다채널 고품질 오디오를 아날로그 변환 과정 없이 하나의 케이블로 다이렉트 지원하며, 고주사율 게이밍 모니터와 그래픽카드 연동에 표준으로 널리 쓰이는 포트 규격은?
정답: DP (DisplayPort)
14. 컴퓨터의 핵심인 CPU 장치의 구조 중 복잡한 다기능 명령어들을 대거 탑재하여 호환성이 뛰어나며, 일반적인 데스크톱 및 고성능 PC 환경의 표준으로 오랫동안 군림해 온 아키텍처 설계 방식은?
정답: CISC (Complex Instruction Set Computer)
15. 메인보드 하드웨어 아키텍처 중, 기존 ATX 규격의 고질적이었던 발열과 통풍 병목 현상을 해결하기 위해 내부 부품의 공기 역학적 일렬(인라인) 배치와 냉각 최적화를 도모했던 확장 폼팩터 표준 규격은?
정답: BTX (Balanced Technology Extended)
16. 비휘발성 플래시 메모리 소자를 탑재하여 탐색 속도가 획기적으로 빠르며, 메인보드에 슬롯 형태로 다이렉트 안착하거나 직렬 고속 확장 인터페이스를 채택해 병목 현상을 없앤 초고속 보조기억장치 규격은?
정답: NVMe SSD
17. 구형 디스플레이와 그래픽카드를 이어주는 아날로그 전용 비디오 영상 출력 인터페이스로, 3열 15핀 구조의 D자형 파란색 플러그 모양을 지닌 레거시 규격 명칭은 무엇인가?
정답: D-Sub (VGA)
18. 데스크톱 컴퓨터 메인보드 및 파워서플라이가 요구하는 전력 분배 배선 규격 중, CPU 전용 전원 제어 및 고성능 프로세서 연산 안정화를 도모하기 위해 메인보드 상단 구석 소켓에 결합하는 보조 전원 커넥터 규격은?
정답: CPU 4+4핀 (8핀) 보조 전원 커넥터

3과목 : PC 주변기기 및 유지보수

19. 사용자가 주소창에 정상적인 인터넷 뱅킹이나 포털 도메인을 정확히 타이핑하여 접속하더라도, PC에 잠입한 악성코드가 DNS 정보를 변조하여 해커가 준비해 둔 정교한 가짜 사기 사이트로 강제 우회 유도한 뒤 금융 기밀 정보를 갈취하는 사이버 범죄 수법은?
정답: 파밍 (Pharming)
20. 본체 전원 스위치를 눌렀을 때 팬은 정상 회전하나 모니터 화면이 전혀 안 들어오며 시스템 내부 메인보드 경고용 스피커에서 비프음이 길게 3회 반복되어 발생할 때, 가장 높은 확률의 고장 원인이 되어 우선 탈거 및 금속 접점 청소를 진행해야 하는 부품은?
정답: RAM (주기억장치 메모리 모듈)
