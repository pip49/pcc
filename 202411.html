PC정비사 2급 필기 기출문제 요약

2024년 11월 03일 시행 | 오개념 방지 교정 버전
1과목: PC유지보수
1. Windows 10 Professional 환경에서 파일 탐색기 및 시스템 UI를 활용하여 특정 디스크 내 임의의 파일을 유효하게 검색하거나 찾아낼 수 있는 정합성 있는 조작 방법들은?
바탕화면에서 [Windows]+[F3] 키 입력, 시작 메뉴의 [프로그램 및 파일 검색] 란 이용, Windows 탐색기 활성화 상태에서 [Ctrl]+[E] 키 또는 [Ctrl]+[F] 키 입력 (※ '내 컴퓨터 우클릭 후 S키'는 시스템 속성창을 띄울 뿐 파일 검색이 실행되지 않음)
2. Windows 10 Professional 시스템에 부팅 결함 등이 발생했을 때 진입할 수 있는 고급 '시스템 복구 옵션' 콘솔에서 엔지니어에게 공식 제공하는 커널 복구 도구는 무엇인가?
시작 복구(자동 복구), Windows 메모리 진단, 시스템 이미지 복구, 명령 프롬프트, 시스템 복원 등 (※ '디스크 오류 복원'이라는 명칭의 단독 옵션 탭은 복구 메뉴에 존재하지 않음)
3. 현대 운영체제의 핵심 아키텍처 기능 중, 사용자에게 시각적·직관적 인터페이스를 제공하는 GUI나 드래그 앤 드롭, 플러그 앤 플레이 및 핫 스왑(Hot swap) 등은 '사용자 편의성'과 직결된다. 그렇다면 이와 달리 하드웨어 자원의 효율성을 극대화하기 위해 백그라운드에서 동작하는 코어 제어 기술은?
다중 작업(Multi-Tasking) 및 다중 스레딩(Multi-Threading) 아키텍처 제어 (※ 이는 편의성보다 자원의 한계 활용 및 처리량 속도 향상을 위한 핵심 메커니즘임)
4. Windows 명령 프롬프트(cmd) 환경에서 메인보드를 탈거하지 않고 물리적으로 장착된 메인보드 칩셋의 정확한 모델명 정보를 커널에 질의하여 알아내기 위한 WMI 표준 명령어 구문은?
wmic baseboard get product
5. 가상화 인프라 기술인 Hyper-V 중, Windows Server 라인업이 아닌 순수 Windows 10 클라이언트 데스크톱 운영체제 환경에서만 독점적·차별적으로 기본 지원되는 하이퍼바이저 네트워크 기능 명칭은?
기본 네트워크 (NAT 스위치)
6. 노트북에 출력 중인 회의 자료 화면을 무선 네트워크 공유 인터페이스 통신을 기반으로 회의실 대형 TV 화면에 실시간 복제 및 송출(무선 디스플레이 연결 캐스팅)하고자 할 때 사용하는 Windows 단축키는?
Windows 로고 키 + K
7. 프로세스 관리의 핵심 데이터로서, 프로세스 실행·스케줄링·메모리 관리 및 파일 입출력 시 수시로 커널에 의해 조회되고 갱신되며, 응용 프로그램이 시스템 호출(System Call)을 통해 내부 정보들을 획득할 수 있는 프로세스 메타 데이터 구조체의 명칭은?
프로세스 제어 블록 (PCB, Process Control Block)
8. Windows 10 환경에서 모니터 해상도, 텍스트 크기, 야간 모드 및 다중 디스플레이 연동 설정을 제어하기 위해 진입해야 하는 메인 [Windows 설정]의 카테고리는?
시스템 (※ 시스템 -> 디스플레이 경로로 진입함)
9. Windows 10 Professional 시스템 파일의 보안 및 관리에 대한 정확한 엔지니어링 지식은 무엇인가?
임의 삭제 방지를 위해 기본적으로 숨김 처리되어 있으며, 오조작 시 커널 크래시가 유발될 수 있어 전용 관리도구 사용이 권장된다. (※ 숨겨진 시스템 파일의 확인 및 편집은 레지스트리를 전혀 변경하지 않고도 폴더 옵션의 숨김 해제 세팅만으로 완벽히 가능하다)
10. 오픈소스 진영의 대표 운영체제인 리눅스(Linux)의 역사와 아키텍처적 계보에 대한 명확한 역사적·공학적 진실은 무엇인가?
유닉스(Unix) 계열의 표준 POSIX 아키텍처 인터페이스를 기반으로 설계되었으며, 1991년 핀란드의 리누스 토르발스(Linus Torvalds)에 의해 커널이 최초로 작성 및 공개되어 무료 사용이 가능하다. (※ 1969년 벨 연구소에서 최초 개발된 것은 리눅스가 아닌 원조 유닉스 운영체제임)
11. 마이크로소프트 Windows 10 운영체제 커널 하에서 외부 서드파티 드라이버의 마운트 없이 기본(Native)적으로 완전 수용하고 제어할 수 있는 표준 파일 시스템 규격 3가지는 무엇인가?
FAT32, NTFS, CDFS, exFAT 등 (※ EXT2/EXT3/EXT4는 리눅스 운영체제의 전용 표준 파일 시스템 규격임)
12. Windows PowerShell 단말 콘솔창을 활용하여 현재 로컬 PC 시스템에 마운트되어 있는 윈도우 OS의 기본 국가 정보 및 지역 언어 세팅 값을 정밀 반환해 주는 쉘 명령어는?
Get-WinSystemLocale
13. Windows 운영체제 커널 패치나 복원 공정 진행 전, 현재 본체에 빌드되어 구동 중인 윈도우의 정확한 메이저 버전, 빌드 번호 및 커널 릴리즈 정보를 시각적 팝업창으로 호출해 주는 실행 명령은?
winver
14. 유닉스 및 리눅스 아키텍처 환경에서 커널 주소 공간에 직접 상주하며 커널 모드로 실행되고, 사용자와 직접적인 I/O 상호 작용을 하지 않으며, 시스템 초기 부팅 시 생성되어 시스템이 전면 종료될 때까지 백그라운드 연산을 전담하는 특권 프로세스의 명칭은?
커널 스레드 (Kernel Thread)
15. Windows 10 제어판에서 제공하는 하드웨어 암호화 도구로, 로컬 드라이브 전체를 고도의 알고리즘으로 래핑 암호화하여 물리적 탈취나 무단 액세스로부터 내부의 모든 파일과 데이터를 정밀 보호하는 보안 기능 명칭은?
BitLocker (비트락커)
2과목: PC운영체제 및 하드웨어
16. 고주사율 게이밍 환경에서 디스플레이의 재생 빈도(Hz)와 그래픽 카드가 렌더링하는 프레임 속도(FPS)를 실시간 동기화하여 화면이 찢어지는 왜곡(Tearing) 현상을 제어하고 입력 지연을 최소화하는 하드웨어 연동 동기화 기술은?
Adaptive Sync Protocol (FreeSync / G-Sync 아키텍처)
17. 디스플레이포트(DP) 출력 단자와 케이블 스택을 사용하여 별도의 분배기 장치 없이 모니터와 모니터를 직렬로 연속 결선함으로써, 하나의 그래픽 출력단으로 여러 대의 멀티 디스플레이 화면을 손쉽게 확장 및 복제 구동하는 기술 명칭은?
데이지체인 (Daisy Chain)
18. 컴퓨터 시스템 아키텍처 분류 중, 외부의 데이터나 물리적 아날로그 사실 정보를 디지털 전기 패킷으로 변환하여 본체 내부 메모리로 밀어 넣어주는 대표적인 물리 입력장치(Input Device) 종류 3가지는?
OMR, OCR, 터치스크린 등 (※ 플로터는 대형 도면이나 그래픽을 물리 종이에 출력해 내는 명확한 '출력장치'임)
19. 3차원 입체 형상을 적층 가공하는 3D 프린터 하드웨어 장치에 대한 공학적 기술 진실은 무엇인가?
인쇄 방식은 재료를 층층이 쌓아 올리는 '적층형' 방식이 대표적이며, 잉크를 평면에 분사하는 일반 2D 프린터와 달리 공간적 입체물을 조형해 낸다
20. Windows 10 디스크 관리 도구에서 복수의 물리 디스크를 결합하여 논리 볼륨을 구성할 때, 물리 디스크 1대에 갑작스러운 결함 및 하드웨어 장애가 터지더라도 패리티 회로망이나 미러링을 통해 데이터 유실 없이 완벽한 자원 복구(결함 허용)가 가능한 볼륨 종류 2가지는?
미러 볼륨 (RAID-1), RAID-5 볼륨 (※ 단순 볼륨, 스팬 볼륨, 스트라이프 볼륨(RAID-0)은 단 1대의 디스크만 고장 나도 전체 데이터가 파괴됨)
21. 시스템 주기억장치 반도체 소자 종류 중, 전원 인입이 중단되어 완전히 차단되더라도 내부의 비트 데이터가 지워지지 않고 영구 보관되는 하드웨어적 대표 '비휘발성 RAM' 규격 종류 3가지는?
PRAM, FRAM, MRAM (※ 컴퓨터 시스템의 메인 메모리로 널리 쓰이는 DRAM은 전원이 차단되면 즉시 데이터가 소멸하는 대표적인 휘발성 메모리임)
22. 중앙처리장치(CPU) 패키지 내부 다이 상에 집적되어 실시간 기계어 연산 루프를 논스톱 처리하는 핵심 물리 내부 하드웨어 구성 요소 3가지는 무엇인가?
레지스터(Register), 제어 유닛(Control Unit), ALU(산술논리연산장치) (※ 버스(Bus)는 이 구성 요소들 간 또는 외부 장치 간에 전기 신호를 전송하기 위해 메인보드 및 칩셋 사이에 배치된 물리적 신호 전선망임)
23. 메인보드 클록 발생기에서 유입되는 메인보드 베이스 클록 주파수 단위를 기반으로 컴퓨터 CPU 소자가 최종적으로 도달하여 동작하는 내부 연산 처리 '최종 CPU 클럭 속도'를 산출해 내는 올바른 공학적 계산 공식은?
시스템 클럭 × 배율 (Clock Multiplier)
24. 디스플레이 장치가 과거 그래픽 표준 규격 인터페이스 환경에서 정합성 있게 화면을 표현하기 위해 지원해 온 역사적·표준적 VGA 모드 화면비 해상도 3가지는?
640×480, 800×600, 1024×768 (※ 275×350 해상도는 표준 VGA 해상도 규격에 존재하지 않는 변칙 수치임)
25. 인텔과 애플이 공동 아키텍처 협업을 통해 제정 및 배포한 초고속 범용 버스 인터페이스로, 구형 USB 규격 대비 압도적인 대역폭을 확보하여 초고속 데이터 전송뿐만 아니라 다이렉트 고해상도 디스플레이 출력 및 전력 전송까지 동시 지원하는 포트 규격 명칭은?
썬더볼트 (Thunderbolt)
26. 백라이트 광원 배치가 필수적인 기존 LCD 디스플레이 장치와 달리, 자체 발광 능력을 가진 유기 화합물 박막층으로 스택을 구성하여 완벽한 블랙 표현이 가능하고 두께를 혁신적으로 줄여 차세대 디스플레이로 널리 쓰이는 출력 장치 소자는?
OLED (유기 발광 다이오드)
27. 고주사율 게이밍 모니터의 제원을 100% 드라이브하기 위해 주로 채택되며, HDMI 규격과 마찬가지로 단일 라인 링크를 통해 디지털 비디오(영상) 신호와 멀티채널 오디오(음성) 신호를 동시에 무손실 송수신하는 고대역폭 표준 인터페이스 규격은?
DP (DisplayPort)
28. 보조기억장치인 하드디스크(HDD) 내부 물리 플래터 미디어가 중심축을 기준으로 스핀들 모터에 의해 1분간 회전하는 물리적 회전 주기를 나타내며, 스토리지의 탐색 속도(Access Time) 성능과 직결되는 단위 명칭은?
RPM (Revolutions Per Minute)
29. 가상 현실(VR) HMD 장치를 착용한 유저가 컴퓨터가 생성해 낸 가상 공간 내부에서 자신의 손 및 상반신의 공간적 좌표 움직임을 실시간 6DoF 센서 체계로 정밀 추적·추종하여 시스템 입력 신호로 변환해 주는 기기는?
모션 컨트롤러 (Motion Controller)
30. 모니터 디스플레이 하드웨어 패널이 백라이트 및 소자를 제어하여 1초 동안 화면 전체에 새롭게 뿌려줄 수 있는 정지 영상의 개수(화면 재생률)를 나타내는 물리 주파수 단위는?
Hz (헤르츠)
3과목: PC주변기기 및 안전관리
31. Windows 10 환경에서 보안 패치 인스톨 꼬임이나 시스템 업데이트 미완료 상태로 인해 커널 불안정이 초래되었을 때, 전산 정비사가 업데이트 레포지토리를 정합성 있게 정상화하기 위해 취해야 할 가장 표준적인 정비 프로세스는 무엇인가?
설정 메뉴 내의 [Windows 업데이트 기록] 콘솔로 진입하여 인스톨 실패나 충돌이 기록된 특정 패치 항목을 명확히 추적해 제거(롤백)한 뒤, 정합성 있는 패치 패키지를 새로 다운로드하여 재설치를 감행한다
